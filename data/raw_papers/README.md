# Raw Papers

This folder is intended to store the initial PDF corpus used in the project.

At this stage, the group has selected an initial subset of 24 open-access papers related to sports injuries of the lower limb. The files may be stored using stable internal names such as `Paper1.pdf`, `Paper2.pdf`, etc.

The bibliographic metadata for these papers should be documented in `data/metadata/papers.csv`.

For the final version of the assignment, the corpus will be expanded to 30 papers.
